# Changelog

### **`v.0.1.1-alpha` from 03.11.2025**
#### Добавлено
- При POST-запросе по маршрутам `/api/v1/generate-from-multiple` и `/api/v1/generate-from-single` теперь в параметрах запроса необходимо указывать `user_id` - уникальный id пользователя.
- Изменены наименования маршрутов. Полные сведения в [README.md](./README_RU.md) 
- Реализован примерный скелет базы данных, основная часть которого реализована в `/api/v1/generate-from-single`. 
- Добавлено сохранение необходимых параметров в соответствующие таблицы базы данных.

#### Изменено
- Обновлены и дополнены параметры при создании запросов к API.
- Доработан и оптимизирован существующий код для повышения стабильности и производительности.
